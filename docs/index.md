## 项目简介
本项目受到 [open-cs-application](https://opencs.app) 的启发, 旨在为**环境科学与工程**及**相近专业**背景的研究生申请者提供一个快速了解可以申请的保研/出国项目难度和特点的平台。您可以在下方评论区提供您的宝贵意见，或者通过 Pull Request 的方式参与贡献。
